# Changelog

## v0.2.0 (2026-07-05)

### 新增

- **多页面 UI**：左侧常驻导航栏，在「日程」和「任务管理」两个页面之间切换
- **任务管理增强**：
  - 模糊搜索（任务名 + 要求字段）
  - 按频率筛选（每天/每周/间隔/单次/指定日期）
  - 按优先级筛选（紧急/重要/一般/可选）
  - 可切换排序（按优先级 / 按名称 / 按创建时间），支持升降序
  - 按优先级排序时自动分组展示（4 桶，空桶不显示）
  - 计数徽章（筛后/总数）
- **多日连续打印**：打印按钮改为下拉菜单
  - 「打印当天」：保留原单日打印逻辑
  - 「打印多日…」：弹窗输入天数（1–31），从当前选中日期起连续 N 天，每天一页合并到一个 PDF
  - 多日 PDF 不处理待定，按原计划时段打印

### 变更

- 应用骨架从单页双栏改为 sidebar + 单页面切换
- DayView 全宽展示（原左栏 1.4fr）
- TaskList 组件被 TaskManage 取代，render_card 抽为共享函数
- `to_print_data` 的 note 字段从空字符串改为携带 task.description（修复多日 PDF 备注列）
- 打印下拉菜单 + 多日弹窗新增样式（亮/暗模式）
- Task 派生 PartialEq（Memo 派生计算需要）
- 字体路径提取为 `font_path_for_os()` 共享函数

## v0.1.0 (2026-07-05)

首个发布版本。

### 核心功能

- **任务管理**：新建、编辑、删除任务，弹出式编辑器
- **多种频率**：每天 N 次 / 每周指定日 / 每 N 天 / 单次 / 日历手动选日期
- **时间段绑定**：每个任务绑定一个或多个时间段（硬绑定），支持无固定时间的"随时"任务
- **优先级**：紧急 / 重要 / 一般 / 可选（4 档）
- **当日打卡表**：按日期生成，按时段排序，无时段任务排末尾
- **冲突检测**：两个任务时段重叠时告警
- **待定标记**：当日视图临时标记任务为"待定"，原位变灰划线 + 末尾追加副本，PDF 只显示末尾
- **打印**：Typst 生成 A4 PDF（中文排版、表格、复选框、复盘区），系统查看器打开
- **任务要求**：每个任务可填执行标准/注意事项，显示在卡片和 PDF 备注列

### 技术栈

- **外壳**：Tauri 2
- **前端**：Leptos 0.8（CSR，WASM）
- **后端**：Rust（tauri commands）
- **存储**：rusqlite（bundled SQLite）+ refinery 迁移
- **打印**：Typst CLI（sidecar）

### 已知限制

- 仅支持 macOS（Apple Silicon + Intel）
- 未签名（需右键打开）
- 打印依赖系统字体（PingFang SC）
